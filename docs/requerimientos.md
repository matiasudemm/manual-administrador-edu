# Requerimientos del sistema

## Sistema operativo
- Windows 10/11 Pro o Windows Server 2019/2022
- Alternativa: Ubuntu Server 22.04 LTS

## Hardware mínimo
- CPU x86_64 con AVX2
- RAM 16 GB (32 GB recomendado)
- GPU NVIDIA (6–8 GB VRAM)
- SSD NVMe (≥ 256–512 GB)

## Software
- Python 3.10.x
- Node.js 18.x LTS
- Docker Engine y Desktop
- Git
- CUDA 11.8 + cuDNN
